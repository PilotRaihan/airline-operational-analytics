# Airline Delay & Operational Performance Analysis

An end-to-end aviation analytics project built using **Python**, **SQL**, and **Power BI** to analyze airline operational performance, flight delays, and airport traffic trends.


## Project Overview

This project analyzes flight operations data to identify delay patterns, airport traffic, airline performance, and operational insights through data cleaning, SQL analysis, and interactive dashboards.

The original dataset contained **3 million flight records**. Due to hardware limitations and to ensure smooth analysis and dashboard performance, the dataset was reduced to **500,000 records** while preserving its overall distribution and analytical value.



## 🛠️ Tech Stack

- Python (Pandas, NumPy)
- MySQL
- Power BI
- CSV Dataset

## 📈 Business Insights

- Southwest Airlines operated the highest number of flights in the dataset.
- ATL (Atlanta) handled the highest flight traffic among origin and destination airports.
- Certain routes consistently experienced significantly higher arrival delays.
- Average departure delay was higher than average arrival delay, indicating operational recovery during flights.
- The dashboard enables quick exploration of airline and airport performance using interactive filters.
